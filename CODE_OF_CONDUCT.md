# Code of Conduct

## Our Pledge
We pledge to make participation in this project a respectful and inclusive experience.

## Our Standards
Be respectful, inclusive, and considerate in communication.

## Enforcement Responsibilities
Project maintainers are responsible for enforcing the code of conduct.

## Scope
This Code of Conduct applies to all project spaces and interactions.

## Enforcement
Violations may result in warnings or removal from the project.

## Enforcement Guidelines
All reports will be reviewed and handled fairly.

## Attribution
This Code of Conduct is adapted from the Contributor Covenant.
